# 接口方法


| Api           | Description                 | Argument          |
| :------------ | :-------------------------- | :---------------- |
| user_login    | 登陆函数                    | account, password |
| get_schedule  | 课表查询                    | year, term        |
| get_score     | 成绩查询                    | year, term        |
| get_info      | 获取个人信息                | None              |
| refresh_info  | 刷新个人信息                | None              |
| check_session | 检查session并其失效后重登录 | None              |
